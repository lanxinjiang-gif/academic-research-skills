# accounting-reviewer-response

A Claude skill for thinking through, designing analyses for, and writing responses to reviewer and Associate Editor (AE) comments at top accounting journals.

**Journals covered:** JAR · TAR · JAE · JOAR · CAR · RAS

## What it does

The skill guides you through three phases:

1. **Analyze** — Categorize each comment by type (framing, methodology, empirical, writing), identify priorities, and decode what reviewers actually want
2. **Design** — Plan the right empirical response: out-of-sample tests, instrumental variables, rolling windows, alternative measures, placebo tests, and more
3. **Write** — Draft polished response memo language in the formal academic register expected at top accounting journals
